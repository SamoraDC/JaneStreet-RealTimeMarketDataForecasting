## What Each Part Does

- `multi-models/run_strong_oof_experiment.py`: CLI entrypoint for the strong OOF
  modular experiment.
- `multi-models/multimodels/strong_oof.py`: implements the Ridge stack,
  target-free nonlinear prediction expansions, batch prediction features
  (`batch_rank`, `batch_mean`, `batch_demean`, `batch_std`, `batch_zscore`),
  fixed blends, gateway audits, scoring, and reports.
- `scripts/run_bayesian_gateway_meta_simulation.py`: builds gateway-style expert
  predictions and feature sets.
- `scripts/run_dynamic_gateway_rls_validation.py`: provides the causal dynamic
  RLS update.
- `scripts/run_gateway_rls_strategy_selection.py`: provides preserved RLS
  strategies and risk-modulated shrinkage candidates used by the strong OOF
  pipeline.
- `src/janestreet/bayesian_meta.py`, `blending.py`, `metrics.py`: shared scoring
  and blending utilities.
- `multi-models/tests/test_strong_oof.py`: focused tests for the strong OOF
  machinery.

## Reproduction Command

The preserved Stage 3 result is generated by the strong OOF experiment with
additional TabM OOF sources, target-free nonlinear prediction expansions, batch
prediction expansions, and a fixed 75/25 blend:

```bash
uv run python multi-models/run_strong_oof_experiment.py \
  --experiment-name strong_oof_subset_s23aux8_s17_gateway_batch_mean_std_stage3_narrow_v1 \
  --output-dir multi-models/reports/strong_oof_subset_s23aux8_s17_gateway_batch_mean_std_stage3_narrow_v1 \
  --extra-prediction-dirs reports/experiments/competitive_tabm_official_stage3_5fold_valid60_lags_online_lr1e4_4m_train700_seed23_aux8_preds/validation_predictions,reports/experiments/competitive_tabm_official_stage3_5fold_valid60_lags_online_lr1e4_4m_train700_seed17_preds/validation_predictions \
  --extra-prediction-prefixes tabm_s23_aux8,tabm_s17 \
  --include-gateway-risk-shrink \
  --include-extra-gateway-experts \
  --gateway-expert-expansions signed_square,cube,pair_product,batch_rank,batch_mean,batch_demean,batch_std,batch_zscore \
  --strong-base-candidates tabm_prediction,tree_prediction,xgboost_prediction,lightgbm_prediction,ridge_calibrated_prediction,baseline_prediction,conservative_rls_prediction,aggressive_rls_prediction,tabm_s23_aux8_prediction,tabm_s17_prediction \
  --fixed-blend-candidates strong_oof_ridge_stack_prediction,gateway_risk_conservative_rls_abs_pred_s100_prediction \
  --fixed-blend-weights 0.75
```

## Notes

This is an OOF/reference implementation. It is not yet a Kaggle runtime
submission package. Making it submissible requires exporting the Ridge stack
coefficients, reproducing batch prediction features at gateway time, reproducing
the RLS features, and applying the final fixed blend inside the runtime.
